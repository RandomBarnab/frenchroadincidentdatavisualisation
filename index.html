🚗 Analyse de la Sécurité Routière en France

Données officielles de l'ONISR (Base BAAC 2021-2024)

Comment utiliser cet outil ?

  1. Paramétrez : Sélectionnez les années ou le type de victimes ci-dessous.
  2. Cadrez : Déplacez la carte sur la zone qui vous intéresse (la zone visible sera analysée).
  3. Analysez : Cliquez sur le gros bouton vert pour générer les résultats.
  4. Filtrez (Magique !) : Maintenez Ctrl (ou Cmd sur Mac) et cliquez sur les éléments des graphiques (ex: "Tué" + "Vélo") pour croiser les filtres dynamiquement sur la carte !
